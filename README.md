jumpstart
=======

Config and boilerplate free dev/build tool.

Installation
------------

#### Global installation
```
npm i -g jumpstart-cli
```

#### Local installation
```
npm i -D jumpstart-cli
```

Usage
-----

#### Globally installed
```
jumpstart
```

#### Locally installed
```
./node_modules/.bin/jumpstart
```

#### Not installed
```
npx jumpstart-cli
```
